Understand Decimal Place Value

  1. Draw a place value chart for the number 9.181. Write its number name and tell the value of the fourth digit.

  2. Write 499.391 in expanded form.

Write each number in standard form

  1. \(\left( 2 \times 1 \right) + \left( 2 \times \frac{1}{100} \right) + \left( 6 \times \frac{1}{1,000} \right)\)

  2. 8 + 0.3 + 0.08 + 0.004

  3. eight hundred thirty-four and ninety-five hundredths

For each question, write two decimals that are equivalent to the given decimal

  1. 6.300
  2. 1.20
  3. 0.500
  4. 2.20

Problem solving

  1. Willdon has a piece of wood that measures \(8 \times \frac{1}{10} + 9 \times \frac{1}{100} + 3 \times \frac{1}{1,000}\) meter. How can this measurement be written as a decimal?

  2. There are 360 people in the movie theater. The same number of people are seated in each of the 6 different sections of the theater. How many people are seated in each section?

  3. Ursola’s softball batting average is 0.950, and Annissa’s is 0.905. Annissa says they have the same average. What error did she make? Explain.

  4. These items are on the menu: Hamburger ($5.45), Salad ($5.95), Sandwich ($5.55), Pizza ($5.05). Juanita spent $11.40. Which two items did Juanita buy?

  5. Regen drew a pentagon with each side measuring 6 inches. Jesse drew a hexagon with each side measuring 9 inches. Which shape has a greater perimeter? Write an equation to help explain your answer.
